Overview

PelHabMSFD is a tool developed by Guillaume Wacquet. It is a desktop GUI application written in R and is distributed as a .tar.gz source release.

PelHabMSFD is designed as a desktop application. It runs on a personal computer (Windows, macOS, or Linux) where a graphical desktop environment is available. The application includes a graphical user interface (GUI), built using R libraries tcltk and tcltk2. These libraries rely on the computers operating system to display windows, buttons and interactive elements.

The software is thus not built on web technologies and cannot not run inside a browser. Also in cloud environments such as remote RStudio containers (e.g. Onyxia - EDITO IDE Rstudio), there is no built-in graphical display and the application can technically run, but its interface cannot be viewed or used interactively (i.e. the software will run but the GUI will not render).
